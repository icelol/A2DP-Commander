# Changelog

Все значимые изменения в проекте A2DP Commander.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [1.2.1] - 2026-01-05

### Добавлено

- **Автоопределение поддержки адаптера** — экспериментальные функции (LDAC Registry, External Encoder) автоматически блокируются если Bluetooth адаптер не поддерживает LDAC/aptX HD
- **Понятные сообщения об ошибках** — при попытке включить неподдерживаемую функцию показывается причина и рекомендации по решению (название адаптера, список рекомендуемых USB трансмиттеров)

### Изменено

- `CanActivateAsync()` в хэндлерах проверяет возможности адаптера перед активацией
- UI блокирует чекбоксы экспериментальных функций для неподдерживаемых адаптеров
- Статус функции показывает причину недоступности вместо общего "Unavailable"

## [1.2.0] - 2026-01-05

### Добавлено

#### Экспериментальные функции
- **LDAC/aptX кодирование через внешний BT-трансмиттер**
  - Rust-сервис encoder (a2dp-encoder.exe) для кодирования аудио
  - Поддержка кодеков: LDAC (990/660/330 kbps), aptX, aptX HD, SBC
  - WASAPI loopback capture для захвата системного аудио
  - Named pipe IPC для управления сервисом из приложения
  - Требует внешний USB Bluetooth трансмиттер (1Mii B03Pro, FiiO BTA30 Pro и др.)

- **Registry Codec Forcing**
  - Экспериментальная функция для изменения настроек кодека через реестр
  - Автоматический backup и restore настроек
  - Требует права администратора и перезагрузку

#### Улучшения стабильности
- **WiFi Coexistence** - отключение конфликта Bluetooth/WiFi на 2.4GHz
- **WiFi Power Saving** - отключение энергосбережения WiFi адаптера для стабильности
- **Processing Period Control** - управление размером буфера через PolicyConfig API
- **Latency Query** - отображение текущей задержки аудио через IAudioClient3

#### UI
- Новая секция "Экспериментальные функции" в главном окне
- Чекбоксы для LDAC Registry и External Encoder
- Настройки кодека и качества для внешнего энкодера
- Tooltips с рекомендованными Bluetooth трансмиттерами
- Локализация на русском и английском языках

### Изменено
- Архитектура FeatureManager для централизованного управления функциями
- Каждая функция имеет механизм rollback для безопасного отключения
- Улучшена диагностика в окне Diagnostics

### Технические детали
- Rust encoder service: ~988 KB (release)
- Поддержка libldac (Apache 2.0) и libopenaptx (GPLv3)
- Build script (build.bat) для сборки обоих компонентов
